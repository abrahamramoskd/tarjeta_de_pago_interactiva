# 💳 Tarjeta de Pago Interactiva

![CSS](https://img.shields.io/badge/CSS-3D_flip-1572b6?style=flat-square&logo=css3&logoColor=white)
![JS](https://img.shields.io/badge/JS-vanilla-f7df1e?style=flat-square&logo=javascript&logoColor=black)
![License](https://img.shields.io/badge/licencia-MIT-6e9ef0?style=flat-square)

Formulario de pago con tarjeta de crédito interactiva en tiempo real. Cada campo que escribes se refleja al instante en la tarjeta — número, nombre, fecha de vencimiento y CVV con flip 3D.

## ✨ Live Demo

👉 **[Ver demo en vivo](https://abrahamramoskd.github.io/tarjeta_de_pago_interactiva/tarjetainteractiva.html)**



## 💥 Características

- 🔢 **Número de tarjeta** — se formatea y muestra en tiempo real
- 👤 **Nombre del titular** — reflejo instantáneo en la tarjeta
- 📅 **Fecha de vencimiento** — actualización dinámica
- 🔐 **CVV con flip 3D** — al enfocar el campo, la tarjeta gira para mostrar el reverso
- ✨ **Animaciones CSS puras** — transición 3D fluida sin librerías

## 🛠️ Tecnologías

- HTML5 semántico
- CSS puro (transform 3D, perspective, transitions)
- Vanilla JS (eventos de input en tiempo real)

tarjeta_de_pago_interactiva/

## 🚀 Uso

Abre `tarjetainteractiva.html` en tu navegador y comienza a escribir en los campos — la tarjeta se actualizará en tiempo real.
